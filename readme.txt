InStream Ad implementation made-easy for publishers.

== Description ==

This plugin implements the Newsmax Feed Network Instream Ad on your website. The Instream ad sits as a discreet link that flows with the core of your site's content, the link is responsive and intelligent. Simply install and seamlessly integrate with the Newsmax Feed Network. All you need is a unique script provided by Newsmax Feed Network support team.

What is the Newsmax Feed Network?
Newsmax Media's unique real-time bidding platform, the Newsmax Feed Network, allows publishers to generate earnings via affiliate links, while synergistically connecting advertisers with websites relevant to the product, service, or content they are promoting.  Our network of advertisers are aggressively bidding for the best placement and exposure across thousands of partnered sites. Every click generated is an earning opportunity.

Whether you are a publisher or an advertiser, if you would like to learn more or join the Newsmax Feed Network, visit https://www.newsmaxfeednetwork.com/ .

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to Plugins > 'Add new' and search for Newsmax, through the 'Plugins' menu in your WordPress site.
2. Install and activate the plugin.
3. Go to Settings > Newsmax InStream Ads to configure your the widget.

== Frequently Asked Questions ==

= How do i see the widget? =

The widget is displayed on single page posts between paragraphs.

= Can i style the widget? =

Yes, you can style the widget to better fit your website style.
